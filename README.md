**🚍 Mini BRT Islamabad Passenger System**

A simple Python console-based application that simulates the basic operations of the Islamabad Bus Rapid Transit (BRT) System. This project was built to practice Python fundamentals such as functions, lists, dictionaries, loops, conditional statements, and user input.

**📌 Features**

**👨‍💼 Admin Panel**

Secure admin login

Add new BRT stations

Add new buses


**👤 User Panel**

View all available stations

View available buses

Search for a station

Check travel fare

Buy a ticket

View purchased ticket
